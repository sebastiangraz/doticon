# AI Icon Demo Site

The Stacks AI icon is a state-machine icon built on a dynamic 3D coordinate system, rendered as SVG. Works as a single state, but supports motion transitions between any two states via vertice interpolation.

`.claude/` To check implementation details and LLM context for future edits.

`npm install && npm run dev` to launch `localhost:3000`
